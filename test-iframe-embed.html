GeoBlock Animation iframe Embed Test

This page tests the GeoBlock animation embedded in an iframe to verify there's no spacing around it.

Test 1: Standard iframe Embed

✓ The gold border should touch all edges of the animation with no gaps
✓ No white/transparent space around the animation
✓ Animation should fill the entire iframe area

Test 2: Smaller Size (400px height)

✓ Even at different heights, no spacing should appear
✓ Animation scales properly

Test 3: Responsive (50% width)

✓ Centered iframe with custom dimensions
✓ No spacing issues at different aspect ratios
